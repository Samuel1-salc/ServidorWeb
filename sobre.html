Sobre o Projeto

Este projeto consiste em um Servidor Web simples em Python desenvolvido com sockets puros.

Características implementadas:

  • Comunicação básica cliente/servidor usando sockets TCP.
  • Recepção e decodificação do protocolo HTTP básico (comando GET).
  • Leitura de arquivos estáticos a partir do disco local.
  • Suporte a múltiplas requisições simultâneas utilizando o módulo threading.
  • Retorno de cabeçalhos de status adequados (200 OK e 404 Not Found).
